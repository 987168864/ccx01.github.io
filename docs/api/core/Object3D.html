[name]

图像对象的基类Object3D。

构造函数

[name]()

该构造函数没有参数。

属性

[property:Integer id]

只读 – 当前对象实例的id。

[property:String uuid]

[link:http://en.wikipedia.org/wiki/Universally_unique_identifier UUID] 当前对象实例的UUID。它是自动分配的，因此不可编辑。

[property:String name]

对象名字（可选，可重复）。

[property:Object3D parent]

对象在图形场景中的父类对象。

[property:Object3D children]

对象在图形场景中的子类对象数组。

[property:Vector3 position]

对象局部位置。

[property:Euler rotation]

对象局部旋转（欧拉角），单位为弧度。

[property:Vector3 scale]

对象局部缩放。

[property:Vector3 up]

向上方向。默认值为THREE.Vector3( 0, 1, 0 )。

[property:Matrix4 matrix]

局部矩阵转换。

[property:Quaternion quaternion]

对象局部四元数旋转[page:Quaternion Quaternion]。

[property:Boolean visible]

值为true时对对象进行渲染。
默认为true。

[property:Boolean castShadow]

被渲染成阴影贴图。
默认为false。

[property:Boolean receiveShadow]

贴图以烘培阴影效果呈现。
默认为false。

[property:Boolean frustumCulled]

当值为true时，对象只有在摄像头视锥内才进行绘制。否则对象总是被绘制，即使它是不可见的。
默认为true。

[property:Boolean matrixAutoUpdate]

当值为true时，它每一帧都会计算位置矩阵（旋转或四元）和缩放，并重新计算matrixWorld属性。
默认为true。

[property:Boolean matrixWorldNeedsUpdate]

当值为true时，它会在当前帧里计算一遍matrixWorld，然后值归为false。
默认为false。

[property:Boolean rotationAutoUpdate]

当值为true时，它每一帧都会计算旋转矩阵。
默认为true。

[property:object userData]

用于存储Object3d自定义数据的对象。它无法被函数直接引用，必须先克隆。

[property:Matrix4 matrixWorld]

对象的全局转换。如果Object3d没有父类对象，那么它等同于局部转换。

方法

[page:EventDispatcher EventDispatcher]方法可作用于这个类。

[method:null applyMatrix]( [page:Matrix4 matrix])

matrix - 矩阵
更新位置，旋转和缩放矩阵。

[method:null translateX]( [page:Float distance] )

distance - 距离
将对象沿着x轴移动。

[method:null translateY]( [page:Float distance] )

distance - 距离
将对象沿着y轴移动。

[method:null translateZ]( [page:Float distance] )

distance - 距离
将对象沿着z轴移动。

[method:Vector3 localToWorld]( [page:Vector3 vector] )

vector - 局部向量
将局部向量转换为全局向量。

[method:Vector3 worldToLocal]( [page:Vector3 vector] )

vector - 全局向量
将全局向量转换为局部向量。

[method:null lookAt]( [page:Vector3 vector] )

vector - 全局指向向量
将对象全局指向目标点。

[method:null add]( [page:Object3D object], ... )

object - 对象
为当前对象添加子对象。可以添加任意个子对象。

[method:null remove]( [page:Object3D object], ... )

object - 对象
为当前对象移除子对象。可以移除任意个子对象。

[method:null traverse]( [page:Function callback] )

callback - 第一个参数为object3D对象的函数
执行当前对象及其所有子类对象上的回调。

[method:null traverseVisible]( [page:Function callback] )

callback - 第一个参数为object3D对象的函数
与traverse作用一样，但是只有可见对象的回调函数会被执行。可见对象的后裔对象回调函数也不执行。

[method:null traverseAncestors]( [page:Function callback] )

callback - 第一个参数为object3D对象的函数
执行当前对象以及所有祖先对象的回调函数。

[method:null updateMatrix]()

更新局部转换。

[method:null updateMatrixWorld]( [page:Boolean force] )

更新对象及其子对象的全局转换。

[method:Object3D clone]()

复制对象及其后裔对象。

[method:Object3D getObjectByName]([page:String name])

name -- 匹配子类对象Object3d.name属性的字符串。
检索对象的子类对象，然后返回第一个匹配到的name。

[method:Object3D getObjectById]([page:Integer id])

id -- 对象id
检索对象的子类对象，然后返回第一个匹配到的id。

[method:Object3D translateOnAxis]([page:Vector3 axis], [page:Float distance])

axis -- 对象空间内的规范化矢量
distance -- 位移距离
在对象空间内沿着轴移动。轴将被当作已规范化。

[method:Object3D rotateOnAxis]([page:Vector3 axis], [page:Float angle])

axis -- 对象空间内的规范化矢量
angle -- 弧度角
在对象空间内沿着轴转动。轴将被当作已规范化。

[method:Array raycast]([page:Raycaster raycaster], [page:Array intersects])

得到当前对象放射线的抽象类。需要Msh,Line,Points类对光线进行投射来实现这个方法。
