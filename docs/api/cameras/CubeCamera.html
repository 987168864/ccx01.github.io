构造函数

[name]([page:number near], [page:number far], [page:number cubeResolution])

near -- 近距离裁切面
far -- 远距离裁切面
cubeResolution -- 设置立方体宽度
构造一个由6个[page:PerspectiveCamera 透视相机]构成的立方体相机，然后渲染到[page:WebGLRenderTargetCube]。

属性

[property:WebGLRenderTargetCube renderTarget]

生成立方体纹理。

方法

[method:null updateCubeMap]([page:WebGLRenderer renderer], [page:Scene scene])

renderer -- 当前WebGL渲染器
scene -- 当前场景
更新renderTarget属性。
