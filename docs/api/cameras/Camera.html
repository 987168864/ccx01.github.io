[name]

摄像头对象的基础类，当你创建一个新的摄像头对象时，这个类将被继承。

构造函数

[name]()

构造函数将matrixWorldInverse和projectionMatrix设定到正确的类型。

属性

[property:Matrix4 matrixWorldInverse]

matrixWorld的逆反矩阵，matrixWorld是摄像头在全局内变换时的矩阵参数。

[property:Matrix4 projectionMatrix]

投影矩阵。

方法

[method:Vector3 getWorldDirection]([page:Vector3 vector])

vector — (可选)

这个方法返回一个表示摄像头在全局内朝向的向量对象。

[method:null lookAt]( [page:Vector3 vector] )

vector — 指向的点

在全局内只要摄像头在场景的(0,0,0)位置，镜头方向就会被指向向量点所在的位置。

[method:Camera clone]( [page:Camera camera] )

camera — 复制摄像头

这个方法返回摄像头的复制对象。
