构造函数

[name]( [page:Float left], [page:Float right], [page:Float top], [page:Float bottom], [page:Float near], [page:Float far] )

left — 相机视锥左侧面。
right — 相机视锥右侧面。
top — 相机视锥上侧面。
bottom — 相机视锥下侧面。
near — 相机视锥近侧面。
far — 相机视锥远侧面。

属性

[property:number zoom]

获取或设置相机的zoom。

[property:Float left]

相机视锥左侧面。

[property:Float right]

相机视锥右侧面。

[property:Float top]

相机视锥上侧面。

[property:Float bottom]

相机视锥下侧面。

[property:Float near]

相机视锥近侧面。

[property:Float far]

相机视锥远侧面。

方法

[method:null updateProjectionMatrix]()

更新相机投影矩阵。参数变更后必须被调用。

[method:OrthographicCamera clone]()


返回正交相机的克隆。

[method:JSON toJSON]()

以JSON格式返回相机数据。
