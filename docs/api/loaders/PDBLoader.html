[name]

.pdb资源加载器。

蛋白质数据库文件格式是描述分子三维结构的文本文件格式。

构造函数

[name]( [page:LoadingManager manager] )

[page:LoadingManager manager] — [page:LoadingManager 加载管理器]。默认为[page:LoadingManager THREE.DefaultLoadingManager]。
创建[name]。

属性

方法

[method:null load]( [page:String url], [page:Function onLoad], [page:Function onProgress], [page:Function onError] )

[page:String url] — 必填。 .pdb文件的URL。
[page:Function onLoad] — 加载完成后调用函数。参数为原子几何图形[page:Geometry geometryAtoms], 键值几何图形[page:Geometry geometryBonds] 的[page:Object JSON]数据。
[page:Function onProgress] — 在加载过程中调用的函数。参数为XmlHttpRequest实例，包含有.[page:Integer total]和.[page:Integer loaded]数据。
[page:Function onError] — 加载错误时将会调用该函数。
从url开始加载，并在onLoad函数中获取返回数据。

[method:Object parsePDB]( [page:String text] )

[page:String text] — 解析pdb结构文本。
解析pdb文本并返回JSON结构数据。

[method:null createModel]( [page:Object json], [page:Function callback] )

[page:Object json] — 解析(JSON) pdb结构数据。
[page:Function callback] — 解析完成后调用，获取3个参数：原子几何图形[page:Geometry geometryAtoms], 键值几何图形[page:Geometry geometryBonds] 以及原始[page:Object json]数据。
解析(JSON) pdb数据并返回两个几何图形[page:Geometry]：原子，键值。

示例

// 加载器实例 var loader = new THREE.PDBLoader(); // 加载PDB资源 loader.load( // 资源URL 'models/molecules/caffeine.pdb', // 资源加载完成函数 function ( geometryAtoms, geometryBonds, json ) { console.log( 'This molecule has ' + json.atoms.length + ' atoms' ); }, // 资源加载中函数 function ( xhr ) { console.log( (xhr.loaded / xhr.total * 100) + '% loaded' ); }, // 资源加载出错函数 function ( xhr ) { console.log( 'An error happened' ); } ); [example:webgl_loader_pdb]
